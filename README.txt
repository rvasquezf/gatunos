Nombre del Módulo Fundamentos del Desarrollo de Páginas Web
Experiencia de Aprendizaje Evaluación N° 1, Fundamentos del Desarrollo de Páginas Web

I.- Estructura HTML
1.- Los datos de contacto son:
a) Rut.
b) Digito verificador.
c) Nombre.
d) Apellido.
e) Dirección.
f) Comuna.

III.- Validaciones de caractéres y su longitud.
1.- Validar los datos requeridos para registrar los clientes:
1) Rut : Solo números 8 dígitos.
2) Digito verificador : Números, k y K. 1 dígito.
3) Nombre : Solo letras. 50 dígitos.
4) Apellido : Solo letras. 50 dígitos.
5) Dirección : Letras, números y espacios. 100 dígitos.
6) Comuna : OMITIR.
